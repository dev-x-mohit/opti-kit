OptiKit

High-performance utilities for modern web applications.

A zero-dependency, isomorphic toolkit optimized for size and execution speed. Compiles to clean ESM and CommonJS, supporting 100% tree-shaking with zero configuration.

Design Philosophy

Tree-Shakeable

Declared side-effects free. Bundlers automatically purge unused helper functions, keeping client bundles microscopic.

Zero Dependencies

Hand-written implementations optimized for security and speed. Zero risk of dependency vulnerability chains.

Dual Modules

Native ES Module (ESM) and CommonJS (CJS) targets generated simultaneously to fit any backend or frontend setups.

Type Safe

Written entirely in TypeScript with auto-generated declaration typings (`.d.ts`) for autocomplete and inline validations.

Quick Start

Integrate OptiKit into your modern web project using your preferred module system or CDN import paths:
